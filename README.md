### 一、相关技术
- 前端：`Vue`、`微信小程序`、`HTML` / `CSS` / `Javascript` 。
- 后端：`Springboot`、`Java`、`JavaWeb` 。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`微信小程序开发工具`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：护工、家属、管理员。

护工功能：1. 登录、2. 注册、3. 首页、4. 床位、5. 公告信息、6. 个人中心

家属功能：1. 登录、2. 首页、3. 个人中心、4. 老人管理、5. 房间管理、6. 床位管理、7. 公告信息管理、8. 健康信息管理、9. 出院申请管理、10. 调动申请管理、11. 护工更换申请管理、12. 老人请假申请管理、13. 轮播图信息

管理员功能：1. 登录、2. 首页、3. 个人中心、4. 管理员管理、5. 家属管理、6. 护工管理、7. 老人管理、8. 房间管理、9. 床位管理、10. 公告信息管理、11. 健康信息管理、12. 床位调动申请管理、13. 出院申请管理、14. 调动申请管理、15. 护工更换申请管理、16. 老人请假申请管理、17. 离职申请管理、18. 请假申请管理、19. 基础数据管理、20. 轮播图信息
